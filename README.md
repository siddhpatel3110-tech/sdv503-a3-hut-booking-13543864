# sdv503-a3-hut-booking-13543864
This is a command-line Hut Booking Manager for the New Zealand Great Walks. 
It allows users to create bookings, validate dates and capacity, view bookings 
for a specific hut and date, cancel bookings, and view occupancy summaries. 
The application stores all data in data.json so bookings persist across restarts.

Run command:
node index.js
